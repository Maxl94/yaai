# YAAI Frontend

Single-page application built with Vue 3, Vuetify, and ECharts for visualizing model monitoring data.

## Stack

- [Vue 3](https://vuejs.org/) (Composition API + TypeScript)
- [Vuetify](https://vuetifyjs.com/) (UI components)
- [ECharts](https://echarts.apache.org/) (charts and visualizations)
- [Vite](https://vite.dev/) (build tooling)

## Development

See the root [CONTRIBUTING.md](../CONTRIBUTING.md) for full setup instructions.

```sh
npm ci
npm run dev
```

The dev server runs on `http://localhost:3000` and proxies `/api` requests to the backend on port 8000.
